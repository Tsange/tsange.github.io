Eastenders - Walford Death Map

Eastenders is a long running British soap drama in the U.K with millions of viewers tuning in weekly to watch. As the show is notorious for killing of characters in dramatic storyline, for a bit I made a map of everywhere a character has died on the set.

Creation

To create the Walford Death Map I used Javascript-based library Leaflet to control the map's main functions such as zooming, adding pins etc.

The actual map element was created in Inkscape using various set photos and layout plans for reference. Although I've tried to be as accurate as possible some features may be missing.
